1. Generate the database
`php artisan migrate`

2. Install backend dependencies:
`composer install`

3. Install frontend dependencies:
`npm install`

4. Start the Laravel server:
`php artisan serve`

5. Start the vue
   -for dev `npm run dev`
   -for build `npm run build`
